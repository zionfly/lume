# Lume

**Осветите свой рабочий процесс с ИИ, который растёт вместе с вами.**

Lume — это десктопный ИИ-ассистент, созданный на базе Tauri v2, с трёхуровневой системой памяти, постепенным раскрытием навыков и интеграцией ботов на нескольких платформах.

## Основные возможности

### 1. Трёхуровневая система памяти
- **L1 (USER.md)**: Профиль пользователя — стиль, предпочтения, часовой пояс (до 1375 символов)
- **L2 (ENV.md)**: Данные об окружении — проекты, соглашения, известные проблемы (до 2200 символов)
- **L3 (SQLite FTS5)**: Полнотекстовый поиск по диалогам, неограниченная история

### 2. Постепенное раскрытие навыков
- **L0**: Только названия и описания навыков (~3K токенов для более чем 40 навыков)
- **L1**: Полная загрузка файла SKILL.md по запросу
- **L2**: Извлечение отдельных разделов
- Автоматическая генерация навыков каждые 15 вызовов инструментов через самооценку

### 3. Продакшн-обвязка
- Изоляция процессов на уровне сессии
- Трассировка вызовов инструментов с длительностью и статусом
- Журналы аудита выполнения в SQLite
- Обнаружение prompt-инъекций

### 4. Шлюз ботов
- Единый адаптерный слой для Feishu, Telegram и DingTalk
- Маппинг слэш-команд на навыки
- Чувствительные операции перенаправляются в десктопное приложение для подтверждения

## Технологический стек

| Уровень | Технология |
|---------|-----------|
| Десктопная оболочка | Tauri v2 |
| Фронтенд | React + TypeScript + TailwindCSS |
| Ядро | Rust (SQLite, Память, Навыки) |
| Агент | Bun Sidecar (маршрутизация LLM, инструменты) |
| ИИ | Anthropic Claude SDK |
| База данных | SQLite с FTS5 |

## Начало работы

```bash
# Установка зависимостей
npm install
cd agent && bun install && cd ..

# Разработка
npm run tauri dev

# Сборка
npm run tauri build
```
