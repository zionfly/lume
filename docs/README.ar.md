# Lume

**أنِر سير عملك بذكاء اصطناعي ينمو معك.**

Lume هو مساعد ذكاء اصطناعي مكتبي مبني على Tauri v2، يتميز بنظام ذاكرة من 3 طبقات، وكشف تدريجي للمهارات، وتكامل مع روبوتات متعددة المنصات.

## الميزات الأساسية

### 1. نظام الذاكرة ثلاثي الطبقات
- **L1 (USER.md)**: ملف المستخدم — الأسلوب، التفضيلات، المنطقة الزمنية (حتى 1375 حرفاً)
- **L2 (ENV.md)**: بيانات البيئة — المشاريع، الاتفاقيات، المشكلات المعروفة (حتى 2200 حرفاً)
- **L3 (SQLite FTS5)**: بحث نصي كامل في المحادثات، سجل غير محدود

### 2. الكشف التدريجي عن المهارات
- **L0**: أسماء المهارات وأوصافها فقط (~3 آلاف رمز لأكثر من 40 مهارة)
- **L1**: تحميل ملف SKILL.md الكامل عند الطلب
- **L2**: استرجاع أقسام محددة
- توليد تلقائي للمهارات كل 15 استدعاء أداة عبر التقييم الذاتي

### 3. إطار الإنتاج
- عزل العمليات لكل جلسة
- تتبع استدعاءات الأدوات مع المدة والحالة
- سجلات تدقيق التنفيذ في SQLite
- كشف حقن الأوامر (Prompt Injection)

### 4. بوابة الروبوتات
- طبقة محول موحدة لـ Feishu و Telegram و DingTalk
- ربط أوامر السلاش بالمهارات
- توجيه العمليات الحساسة إلى تطبيق سطح المكتب للموافقة

## المكدس التقني

| الطبقة | التقنية |
|--------|---------|
| غلاف سطح المكتب | Tauri v2 |
| الواجهة الأمامية | React + TypeScript + TailwindCSS |
| النواة | Rust (SQLite، الذاكرة، المهارات) |
| الوكيل | Bun Sidecar (توجيه LLM، الأدوات) |
| الذكاء الاصطناعي | Anthropic Claude SDK |
| قاعدة البيانات | SQLite مع FTS5 |
