# Lume

**あなたと共に成長する AI で、ワークフローを照らし出す。**

Lume は Tauri v2 で構築されたデスクトップ AI アシスタントです。3層メモリシステム、段階的スキル開示、マルチプラットフォームボット連携を備えています。

## 主な機能

### 1. 3層メモリシステム
- **L1 (USER.md)**：ユーザープロファイル — スタイル、好み、タイムゾーン（1375文字以内）
- **L2 (ENV.md)**：環境情報 — プロジェクト、規約、既知の問題（2200文字以内）
- **L3 (SQLite FTS5)**：全文会話検索、無制限の履歴

### 2. 段階的スキル開示
- **L0**：スキル名と説明のみ（40以上のスキルで約3Kトークン）
- **L1**：必要に応じて完全な SKILL.md を読み込み
- **L2**：特定セクションの取得
- 15回のツール呼び出しごとに自己評価を通じてスキルを自動生成

### 3. プロダクションハーネス
- セッションごとのプロセス分離
- 実行時間・ステータス付きのツール呼び出しトレース
- SQLite に実行監査ログを記録
- プロンプトインジェクション検出

### 4. Bot ゲートウェイ
- Feishu、Telegram、DingTalk 向け統一アダプターレイヤー
- スラッシュコマンドからスキルへのマッピング
- 機密操作はデスクトップアプリに転送して承認

## 技術スタック

| レイヤー | 技術 |
|----------|------|
| デスクトップシェル | Tauri v2 |
| フロントエンド | React + TypeScript + TailwindCSS |
| コア | Rust（SQLite、メモリ、スキル） |
| エージェント | Bun Sidecar（LLMルーティング、ツール） |
| AI | Anthropic Claude SDK |
| データベース | SQLite + FTS5 全文検索 |

## はじめに

```bash
# 依存関係のインストール
npm install
cd agent && bun install && cd ..

npm run tauri dev

# ビルド
npm run tauri build
```
